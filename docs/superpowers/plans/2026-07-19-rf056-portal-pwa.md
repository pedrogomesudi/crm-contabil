# RF-056 — Portal do cliente como PWA instalável — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** tornar o portal do cliente (`(portal)`) um PWA instalável — manifest escopado a `/portal`, ícones, service worker mínimo com fallback offline — sem cache offline de dados.

**Architecture:** `app/manifest.ts` (scope `/portal`) + `metadata`/`viewport` no layout do portal + `public/sw.js` mínimo + rota `/portal/offline` + `RegistrarServiceWorker` client no layout + ícones em `public/icons/`.

**Tech Stack:** Next 16 (App Router, `MetadataRoute.Manifest`), TypeScript, service worker vanilla, Node `zlib` para gerar PNGs. **Sem migration.**

## Global Constraints

- Next 16: `middleware.ts` é `proxy.ts`; imports `@/*`; imagens em app usam `next/image` (ícones do PWA são assets em `public/`, não `<img>`).
- Guard `rotas-alcancaveis` cobre `app/(app)` + `components` — o portal (`app/(portal)`) tem layout próprio e **fica fora** desse guard (por isso `/portal/*` já existe sem entrar em `POR_SUBNAV`). Confirmar rodando o guard.
- Guard `divida-ui`: sem controles de formulário nesta fatia (nada a violar).
- `scripts/*.mjs`: JS puro, fora do `tsc --noEmit`, cobertos por ESLint. O gerador de ícones vive aqui.
- Rodar antes de entregar: `lint`, `typecheck`, `test`, `format`, `build`. PR `develop`→`main`; tag após deploy; versão+CHANGELOG no mesmo PR. **Sem migration** → nada a aplicar no banco.

- [ ] **Step 1: Escrever o gerador**

Usa `zlib.deflateSync` + `zlib.crc32` (Node ≥ 22) para encodar PNG RGBA; desenha fundo verde da marca (`#0FA968`) e um "S" branco a partir de um bitmap 7×7 escalado. O maskable usa mais padding (safe zone).

```js
// Gera os ícones do PWA do portal (verde da marca + "S" branco). PNG sem dependências.
import { deflateSync, crc32 } from "node:zlib";
import { writeFileSync, mkdirSync } from "node:fs";

const VERDE = [0x0f, 0xa9, 0x68];
const BRANCO = [0xff, 0xff, 0xff];

// "S" 7x7 (1 = branco).
const S = [
  "0111110",
  "1000000",
  "1000000",
  "0111110",
  "0000001",
  "0000001",
  "0111110",
].map((r) => r.split("").map(Number));

function u32(n) {
  const b = Buffer.alloc(4);
  b.writeUInt32BE(n >>> 0, 0);
  return b;
}
function chunk(type, data) {
  const t = Buffer.from(type, "latin1");
  const body = Buffer.concat([t, data]);
  return Buffer.concat([u32(data.length), body, u32(crc32(body))]);
}
function png(size, glyphFrac) {
  // pixels RGBA
  const raw = Buffer.alloc(size * (size * 4 + 1)); // +1 filtro por linha
  const cell = Math.floor((size * glyphFrac) / 7);
  const gw = cell * 7;
  const ox = Math.floor((size - gw) / 2);
  const oy = Math.floor((size - gw) / 2);
  for (let y = 0; y < size; y++) {
    const rowStart = y * (size * 4 + 1);
    raw[rowStart] = 0; // filtro None
    for (let x = 0; x < size; x++) {
      const gx = Math.floor((x - ox) / cell);
      const gy = Math.floor((y - oy) / cell);
      const branco = gx >= 0 && gx < 7 && gy >= 0 && gy < 7 && S[gy][gx] === 1;
      const [r, g, b] = branco ? BRANCO : VERDE;
      const p = rowStart + 1 + x * 4;
      raw[p] = r;
      raw[p + 1] = g;
      raw[p + 2] = b;
      raw[p + 3] = 255;
    }
  }
  const ihdr = Buffer.concat([u32(size), u32(size), Buffer.from([8, 6, 0, 0, 0])]); // 8-bit RGBA
  const sig = Buffer.from([137, 80, 78, 71, 13, 10, 26, 10]);
  return Buffer.concat([
    sig,
    chunk("IHDR", ihdr),
    chunk("IDAT", deflateSync(raw)),
    chunk("IEND", Buffer.alloc(0)),
  ]);
}

mkdirSync("public/icons", { recursive: true });
writeFileSync("public/icons/portal-192.png", png(192, 0.62));
writeFileSync("public/icons/portal-512.png", png(512, 0.62));
writeFileSync("public/icons/portal-maskable-512.png", png(512, 0.5)); // mais padding p/ safe zone
console.log("ícones gerados em public/icons/");
```

- [ ] **Step 5: Service worker `public/sw.js`**

```js
const OFFLINE = "/portal/offline";
self.addEventListener("install", (e) => {
  e.waitUntil(caches.open("portal-v1").then((c) => c.add(OFFLINE)));
  self.skipWaiting();
});
self.addEventListener("activate", () => self.clients.claim());
self.addEventListener("fetch", (e) => {
  // Só navegações (HTML): se a rede cair, mostra a página de offline. Nada de dados é cacheado.
  if (e.request.mode === "navigate") {
    e.respondWith(fetch(e.request).catch(() => caches.match(OFFLINE)));
  }
});
```

- [ ] **Step 1: Componente de registro**

```tsx
"use client";
import { useEffect } from "react";

export function RegistrarServiceWorker() {
  useEffect(() => {
    if ("serviceWorker" in navigator) {
      navigator.serviceWorker.register("/sw.js", { scope: "/portal" }).catch(() => {});
    }
  }, []);
  return null;
}
```
