- **Só visual:** preservar `name`/`value`/`onChange`/actions/`aria`/`role`; cálculos, RLS e comportamento inalterados.
- **Mapa de tokens (mesmo da V8.2a):** `text-slate-900→text-texto`; `text-slate-700/600→text-cinza`; `text-slate-500/400→text-cinza-claro`; `border-slate-*→border-linha`; `divide-slate-100→divide-linha/70`; `bg-slate-100/50→bg-creme`; `hover:bg-slate-50→hover:bg-creme`; botão primário `bg-slate-900`→`bg-verde` (`rounded-lg … font-medium hover:brightness-105`); `text-red-600/700→text-negativo`; `bg-red-50→bg-negativo/10`; `text-green-700→text-verde`; `bg-green-50→bg-verde/10`; `text-amber-800/bg-amber-100`→manter (semântico "atenção"); títulos `text-2xl font-semibold`→`font-display text-2xl font-bold tracking-tight`; valores monetários→`font-mono`.
- **Guia:** `docs/design/saldo-ui.md`. **Semântica:** positivo=verde, negativo=negativo, atenção=âmbar, dados=violeta/neutro.

```ts
// Status do título → variante de cor do Badge.
export function badgeStatusTitulo(status: string): "positivo" | "atencao" | "negativo" | "neutro" {
  const s = (status ?? "").toUpperCase();
  if (s === "BAIXADO") return "positivo";
  if (s === "BAIXADO_PARCIAL") return "atencao";
  if (s === "VENCIDO") return "negativo";
  return "neutro";
}
```

- [ ] **Step 4: Rodar (passa)**

Run: `npm test -- src/tests/ui/apresentacao.test.ts`
Expected: PASS.
